# Math Visual Generator

Math Visual Generator is a Python-based application for creating accurate
mathematical graphs and diagrams for Grade 8 to Grade 12 Pure Mathematics.

## Project Goal

The goal of this project is to make it easier to generate clear, accurate and
reusable mathematical visuals for lessons, exercises, assessments and study
materials.

## Planned Features

- Linear function graphs
- Quadratic function graphs
- Exponential and hyperbolic graphs
- Trigonometric graphs
- Coordinate geometry diagrams
- Euclidean geometry diagrams
- Statistical graphs
- Probability diagrams
- Transformations
- Exporting visuals as PNG, SVG and PDF files

## Technologies

- Python
- Matplotlib
- NumPy
- SymPy
- Plotly
- Pillow
- NetworkX
- Manim

## Current Development Stage

The project is currently in its initial development stage. The first feature
will be a quadratic graph generator that identifies and displays:

- x-intercepts
- y-intercepts
- turning points
- axes of symmetry

## Target Users

- Mathematics teachers
- Mathematics content creators
- Tutors
- Grade 8–12 learners
- Educational software developers
